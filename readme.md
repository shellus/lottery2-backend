# lottery2-backend

彩票项目第二版本的PHP后端代码，包括彩票种类、玩法信息、期数规则、开奖结果采集和储存、投注、计算中奖金额等等。。。

## 使用方法

```bash

git clone git@github.com:shellus/shcms.git
cd shcms

cp .env.example .env
#编辑 .env 文件，修改数据库连接信息

composer install

php artisan migrate
php artisan db:seed

```
